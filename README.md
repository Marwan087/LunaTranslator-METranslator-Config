# LunaTranslator Integration for METranslator

This repository contains a customized configuration script (`selfbuild.py`) that enables seamless integration between **METranslator** and **LunaTranslator**.

**METranslator** is a powerful, privacy-focused, and local-first AI translation tool. This configuration script allows **LunaTranslator** (a popular visual novel translator) to recognize and communicate with your local METranslator server, enabling high-quality offline translations directly within your games.

## ðŸš€ Installation

1.  **Download** the `selfbuild.py` file from this repository.
2.  **Locate** your LunaTranslator installation directory.
3.  **Navigate** to the `userconfig` folder (e.g., `LunaTranslator/userconfig`).
4.  **Place** the `selfbuild.py` file into this folder.
5.  **Restart** LunaTranslator.
6.  Open LunaTranslator Settings, go to the **Translator** tab, and select **Custom Translation** as your usage source.
